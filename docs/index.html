geographical Location

http://purl.org/voc/hp/geographicalLocation

Description Annotations Relations

The geographical location of a place related to some other place. This can be used for denoting the location of historic villages where exact position information are not known but a relation to some well known place exists. E.g. “North of Leipzig”.
